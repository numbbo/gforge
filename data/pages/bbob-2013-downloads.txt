This is the BBOB 2013 download page (v13.09). We encourage submission with small budgets on the noise-free testbed (f1-f24). On the noisy testbed use the not-expensive option for the Python postprocessing. 

[[http://coco.lri.fr/downloads/download13.09/bbobexp13.09.tar.gz|BBOB (5MB)]] is all that is needed to run the benchmarking experiments and compile a workshop paper (gathering post-processed results). 

[[http://coco.lri.fr/downloads/download13.09/bboball13.09.tar.gz|BBOB (35MB)]] contains all files, as listed below. 

  * CODE:
    *[[http://coco.lri.fr/downloads/download13.09/bbobmatlab.tar.gz|tar]] code in Matlab/Octave to run experiments
    *[[http://coco.lri.fr/downloads/download13.09/bbobc.tar.gz|tar]] code in C to run experiments
    *[[http://coco.lri.fr/downloads/download13.09/bbobjava.tar.gz|tar]] code in Java to run experiments
    *[[http://coco.lri.fr/downloads/download13.09/bbobpproc.tar.gz|tar]] code in Python to run experiments and post-processing and latex templates (3MB)
    *[[http://coco.lri.fr/downloads/download13.09/bbobr.tar.gz|tar]] code in R to run experiments

  * Latex templates for the workshop:
      *[[http://coco.lri.fr/downloads/download13.09/bbobworkshoptemplates.tar.gz|tar]] BBOB specific latex templates

  * PDF DOCS:
    *[[http://coco.lri.fr/downloads/download13.09/bbobdocexperiment.pdf|pdf]] description of experimental procedure  
    *[[http://coco.lri.fr/downloads/download13.09/bbobdocfunctions.pdf|pdf]] (12MB) noiseless functions documentation with figures
    *[[http://coco.lri.fr/downloads/download13.09/bbobdocfunctionsdef.pdf|pdf]] noiseless functions documentation, version without figures
    *[[http://coco.lri.fr/downloads/download13.09/bbobdocnoisyfunctions.pdf|pdf]] (20MB) noisy function documentation with figures      
    *[[http://coco.lri.fr/downloads/download13.09/bbobdocnoisyfunctionsdef.pdf|pdf]] noisy function documentation, version without figures  
    *[[http://coco.lri.fr/downloads/download13.09/bbobdocsoftware.pdf|pdf]] software user documentation  

  * ONLINE DOCS:
    *[[http://coco.lri.fr/COCOdoc|html online]] post-processing package documentation and more
