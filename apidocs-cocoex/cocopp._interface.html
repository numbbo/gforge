This package is meant to generate output figures and tables for the benchmarking of continuous optimisers in the case of black-box optimisation. The post-processing tool takes as input data from experiments and generates outputs that will be used in the generation of the LateX- formatted article summarizing the experiments.

The main method of this package is cocopp.main (currently aliased to cocopp.rungeneric.main). This method allows to use the post-processing through a command-line interface.

To obtain more information on the use of this package from the python interpreter, type help(cocopp.cococommands), however remark that this info might not be entirely up-to-date.
