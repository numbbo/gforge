bbob_pproc API Documentation

Module Index

  • bbob_pproc - COmparing Continuous Optimisers (COCO) post-processing software
    • bbob_pproc.__main__ - python bbob_pproc tests the package bbob_pproc and should run through smoothly from a system command shell. It however depends on data files that might not be available (to be improved).
    • bbob_pproc.algportfolio - Algorithm portfolio data set module.
    • bbob_pproc.bestalg - Best algorithm dataset module
    • bbob_pproc.bwsettings - This module contains settings for outputting black and white figures.
    • bbob_pproc.changeAlgIdAndComment - No module docstring; 0/1 classes, 1/2 functions documented
    • bbob_pproc.cococommands - Module for using COCO from the (i)Python interpreter.
    • bbob_pproc.comp2 - COmparing Continuous Optimisers (COCO) post-processing tool for comparing two algorithms:
      • bbob_pproc.comp2.ppfig2 - Creates aRT-ratio comparison figures (ECDF) and convergence figures for the comparison of 2 algorithms.
      • bbob_pproc.comp2.pprldistr2 - Creates run length distribution figures for the comparison of 2 algorithms.
      • bbob_pproc.comp2.ppscatter - Scatter Plots.
      • bbob_pproc.comp2.pptable2 - Rank-sum tests table on "Final Data Points".
    • bbob_pproc.compall - COmparing Continuous Optimisers (COCO) post-processing tool for comparing multiple algorithms:
      • bbob_pproc.compall.determineFtarget2 - No module docstring; 1/1 classes, 4/5 functions documented
      • bbob_pproc.compall.organizeRTDpictures - Routine to organize the output folder in which runcompall puts all the output.
      • bbob_pproc.compall.ppfigs - Creates aRTs and convergence figures for multiple algorithms.
      • bbob_pproc.compall.ppperfprof - For generating performance profiles. This module is not in use (anymore or not yet?).
      • bbob_pproc.compall.pprldmany - Generates figure of the bootstrap distribution of aRT.
      • bbob_pproc.compall.pptables - Routines for the generation of TeX tables.
    • bbob_pproc.config - This module is an attempt for a global configuration file for various parameters.
    • bbob_pproc.dataoutput - Routines for outputting python-formatted data.
    • bbob_pproc.determineFtarget - No module docstring; 2/2 classes, 4/5 functions documented
    • bbob_pproc.findfiles - Recursively find :file:`info` and :file:`pickle` files within a directory
    • bbob_pproc.firstsession - First session script.
    • bbob_pproc.genericsettings - This module contains some variables settings for COCO.
    • bbob_pproc.grayscalesettings - This module contains settings for outputting grayscale figures.
    • bbob_pproc.htmldesc - Prepares the descriptions of images and tables which will be converted to html.
    • bbob_pproc.old_ranksumtest - Stats including rank-sum test for small sample and correct tied ranks. Author: Sturla Molden http://mail.scipy.org/pipermail/scipy-user/2009-February/019759.html
    • bbob_pproc.ppconverrorbars - Process data and generates some comparison results.
    • bbob_pproc.ppfig - Generic routines for figure generation.
    • bbob_pproc.ppfigdim - Generate performance scaling figures.
    • bbob_pproc.ppfigparam - Generate aRT vs param. figures.
    • bbob_pproc.pplogloss - Module for computing aRT loss ratio
    • bbob_pproc.pprldistr - For generating empirical cumulative distribution function figures.
    • bbob_pproc.pprldistr2009_hardestRLB - Generates "pprldistr2009_RLB.pickle.gz" that is used to plot, in the background, the results of the 2009 algorithms
    • bbob_pproc.pproc - Raw post-processing routines.
    • bbob_pproc.pprocold - Helper routines for read index files.
    • bbob_pproc.ppsingle - Single data set results output module.
    • bbob_pproc.pptable - Module for generating tables used by rungeneric1.py.
    • bbob_pproc.pptex - Routines for writing TeX for tables.
    • bbob_pproc.preparehtml - Prepares the figure and table descriptions in html.
    • bbob_pproc.preparetexforhtml - Prepares the descriptions of images and tables which will be converted to html.
    • bbob_pproc.readalign - Helper routines to read in data files.
    • bbob_pproc.rungeneric - Process data to be included in a latex template.
    • bbob_pproc.rungeneric1 - Module for post-processing the data of one algorithm.
    • bbob_pproc.rungeneric2 - Routines for the comparison of 2 algorithms.
    • bbob_pproc.rungenericmany - Process data to be included in a generic template.
    • bbob_pproc.sanitycheck - Module for checking data sets.
    • bbob_pproc.testbedsettings - No module docstring; 5/5 classes, 0/3 functions documented
    • bbob_pproc.toolsdivers - Various tools.
    • bbob_pproc.toolsstats - Bootstrapping and statistics routines.
