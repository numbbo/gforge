Average number of (f+g)-evaluations to reach target

Scaling of runtime with dimension to reach certain target values ∆f. Lines: average runtime (aRT); Cross (+): median runtime of successful runs to reach the most difficult target that was reached at least once (but not always); Cross (×): maximum number of (f+g)-evaluations in any trial. Notched boxes: interquartile range with median of simulated runs; All values are divided by dimension and plotted as log10 values versus dimension. Shown is the aRT for fixed values of ∆f = 10k with k given in the legend. Numbers above aRT-symbols (if appearing) indicate the number of trials reaching the respective target. Horizontal lines mean linear scaling, slanted grid lines depict quadratic scaling.
